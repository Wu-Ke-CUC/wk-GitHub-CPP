# 三维载具漫游与相机控制 - Visual C++ OpenGL 版本

基于 Three.js 网页程序，使用 Visual C++ 和 OpenGL 完美复刻的 Win32 窗口程序。

## 功能特性

### 场景元素
- 草地平面（草绿色）
- 网格辅助线
- 30 棵随机分布的树木
- 5 座远景山脉
- 天空球背景

### 载具控制（方向键）
- `↑` 前进
- `↓` 后退
- `←` 左转
- `→` 右转

### 相机控制（WSAD键）
- **跟随模式**：`W/S/A/D` 控制相机视角偏移
- **自由模式**：`W/S/A/D` 控制相机位置移动

### 功能按钮
- **重置相机 [F2]**：恢复相机默认位置和视角
- **切换模式**：在跟随模式和自由模式之间切换

### 状态显示
- 载具位置（X, Y, Z）
- 载具速度
- 相机角度
- 当前相机模式

## 构建说明

### 环境要求
- Windows 10/11
- Visual Studio 2022 或更高版本
- OpenGL 支持

### 编译步骤

1. 使用 Visual Studio 打开 `Vehicle3D.sln`
2. 选择 `Debug` 或 `Release` 配置
3. 选择 `Win32` 平台
4. 按 `Ctrl+Shift+B` 编译项目
5. 按 `F5` 运行程序

### 编译选项
- **Debug**：包含调试信息，适合开发调试
- **Release**：优化编译，适合发布使用

## 操作说明

1. 程序启动后，相机自动处于跟随载具模式
2. 使用方向键控制载具移动和转向
3. 使用 WSAD 键调整相机视角
4. 点击按钮或按 F2 重置相机
5. 按 ESC 退出程序

## 技术实现

- **渲染**：OpenGL 1.1 + GLU
- **窗口**：Win32 API
- **数学**：自定义 3D 数学库
- **输入**：Win32 消息处理
- **UI**：GDI 文字渲染 + OpenGL 图形
